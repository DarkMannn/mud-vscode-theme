## Installation

1. Open the **Extensions** sidebar in VS Code. `View → Extensions`
1. Search for `Mud Theme`, choose "Mud Theme" by **DarkMannn**
1. Click **Install** to install it
2. Navigate to File > Preferences > Color Theme > **Mud Theme**
